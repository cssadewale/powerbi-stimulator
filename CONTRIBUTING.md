# Contributing

Thank you for improving PowerBI Stimulator Enterprise.

## Contribution Areas
- More visual types using SVG/canvas without paid APIs.
- Better XLSX support using free optional libraries.
- Web Worker parsing for very large CSV files.
- More DAX-style helper functions.
- Better accessibility and keyboard navigation.
- Additional governance templates for schools and SMEs.

## Rules
1. Do not add paid AI APIs.
2. Do not require a backend for the core app.
3. Keep the app usable as a static site.
4. Preserve existing learning features.
5. Document every new feature in `FEATURES.md` and `docs/user-guide.md`.

## Workflow
1. Fork the repository.
2. Create a branch: `feature/your-feature`.
3. Make changes.
4. Test by opening `index.html` locally.
5. Submit a pull request with screenshots and a clear explanation.
