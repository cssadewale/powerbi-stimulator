# User Guide

## 1. Load Data
Use **Get data** or **Sample**. CSV and JSON run offline. XLSX users can export Excel as CSV or integrate free SheetJS.

## 2. Review Data
Open **Data View**, click a dataset and check the dictionary and preview.

## 3. Profile Data
Open **Power Query / Statistics**, choose a dataset and click **Compute**.

## 4. Build a Report
Open **Report View**, add a visual and select dataset/category/value fields.

## 5. Create a Pivot
Open **Pivot**, select row, column, value and aggregation fields, then click **Build**.

## 6. Join Tables
Open **Model View**, select left and right datasets, keys and join type, then click **Run join**. Save the joined result if needed.

## 7. Add a Calculated Field
Open **DAX Measures**, select dataset, enter a field name and formula, test it, then apply.

## 8. Use Enterprise Controls
Open **Enterprise Center** to select a role, review governance checks, save versions, add validation rules, export audit logs and create backups.

## 9. Publish or Backup
Use **Publish** to download `.pbix.json`. Use **Backup** to download a restorable workspace file.
