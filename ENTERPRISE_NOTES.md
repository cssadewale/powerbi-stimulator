# Enterprise Enhancement Notes

This folder contains the enterprise edition of PowerBI Stimulator.

The platform is a **Power BI-style enterprise simulator** for learning, demonstrations and portfolio use. It is not affiliated with Microsoft and does not copy Microsoft source code. It recreates the familiar BI concepts, layouts and workflows using free static web technologies.

## Enterprise Features Added

- Workspaces
- Semantic model relationship discovery
- Row-Level Security simulation
- Data lineage view
- Refresh and gateway simulation
- Deployment pipeline simulation
- App publishing and sharing simulation
- Tenant administration and audit logs
- Sensitivity labels and endorsement simulation
- Metric alerts and subscriptions
- Bookmark snapshots
- Mobile report layout preview
