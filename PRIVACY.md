# Privacy Notice

PowerBI Stimulator Enterprise is privacy-first by design.

## Data Processing
Imported files are parsed in the browser. The app does not upload files to a server.

## Local Storage
Workspace state may be stored in browser local storage so users can continue later on the same device.

## User Responsibility
Users should not import sensitive personally identifiable information into public or shared devices. For classroom demonstrations, use the included sample data or anonymised data.

## AI API Position
The system does not use AI APIs because they are not cost-effective for the intended use case.
